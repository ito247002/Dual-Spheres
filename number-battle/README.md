# Number Battle - 弾幕シューティング

YouTube Shorts向けの東方Project風2D弾幕バトルシミュレーション

## 概要

2つの数字オブジェクトが弾を撃ち合い、HP=0で爆発するバトル演出。

## 仕様

| 項目 | 値 |
|------|-----|
| 画面サイズ | 9:16 (YouTube Shorts用) |
| 初期HP | 100 |
| 弾ダメージ | 1 |
| 発射間隔 | 100ms |
| 同時発射数 | 6発 |

## 特徴

- **弾幕演出**: 扇状に複数弾を発射
- **壁反射**: オブジェクトは壁でバウンド、弾は壁で消滅
- **ノックバック**: 被弾時にオブジェクトが反射
- **爆発演出**: HP=0でパーティクル爆発
- **永続アニメーション**: ブラウザを閉じるまで動作継続
- **パステルカラー**: ピンク vs ミント

## ファイル構成

number-battle/

## 起動方法

```bash
npx -y http-server . -p 8080 -c-1
```

ブラウザで http://127.0.0.1:8080 にアクセス

## 動画化

1. ブラウザでゲームを開く
2. OBS等で画面録画
3. YouTube Shortsにアップロード
